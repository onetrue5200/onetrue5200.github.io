第2章 变量和基本类型

2.1 基本内置类型

算数类型

  • 布尔
    • bool 取值为真或假
  • 字符
    • char 确保能存放机器基本字符集
    • wchar_t 确保能存放机器最大扩展字符集
    • char16_t/char32_t 为Unicode字符集服务
  • 整型
    • shot
    • int
    • long
    • long long
  • 浮点型
    • float
    • double
    • long double

整型和char可以加 unsigned表示无符号类型。无符号类型要注意运算时的边界问题和自动转化问题。

字面值常量

字面值常量指形如42的直接书写的值。

整型字面值可以写作十进制(20)、八进制(024)或十六进制(0x14)。整型字面值具体数据类型由它的值和符号决定，我们也可以通过前缀或后缀改变字面值的数据类型。

浮点型字面值可以写作小数(3.14)或科学计数法(1e5)。

由单引号括起来的一个字符称为char型字面值，双引号括起来的多个字符称为字符串型字面值。

true和false是bool类型的字面值，nullptr是指针字面值。

2.2 变量

变量定义的基本形式，首先类型说明符，后接变量名列表。

变量在创建时需要初始化才能使用，只有全局变量才会被默认初始化。

为了支持分离式编译，C++将声明和定义区分开，声明使得名字为程序所知，一个文件如果想使用别处定义的名字则必须包含对那个名字的声明。而定义负责创建于名字相关联的实体。extern关键字用于声明一个变量。

变量只能被定义一次，但可以被多次声明。

作用域是程序的一部分，在其中名字有其特定的含义。函数体外的的名字拥有全局作用域，在整个程序范围内都可以使用。花括号内的名字仅在该区域内可以访问，访问符合就近原则。

2.3 复合类型

引用

引用是给对象起别名，在定义是必须初始化，把引用于初始值绑定在一起，引用只能绑定在对象上，不能绑定字面值。

因为引用本身不是对象，所以不能定义引用的引用。

指针

指针存储地址的对象，用于指向其他对象，无需在定义时初始化。

定义指针时最好初始化为 nullptr。

2.4 const限定符

给变量加上 const限定符，该变量的值不能被改变。

对常量的引用也必须是常量 const int &r = a。引用常量可以引用非常量，这样通过非常量也可以修改值。

指向常量的指针不能用于改变对象的值。想要存放常量对象的地址，只能使用指向常量的指针。

指针是对象，允许把指针本身定为常量，表示地址不再改变。

常量表达式是指不会改变并且在编译过程就能得到计算结果的表达式。constexpr限定符可以验证变量的值是否是一个常量表达式。
